## 📝 归档原则

### 保留原则

文档满足以下任一条件即保留：
- ✅ 包含独特技术方案或问题解决方法
- ✅ 记录重要架构决策或演进历程
- ✅ 具有长期参考价值
- ✅ 可能被未来项目复用

### 淘汰原则

文档满足以下任一条件即淘汰：
- ❌ 内容完全过时，无参考价值
- ❌ 已被更完善的文档完全替代
- ❌ 临时性文档，已完成使命
- ❌ 长期（6个月+）未被访问

### 归档策略

- 📁 **活跃归档**: 当前分类目录（频繁访问）
- 📦 **深度归档**: 创建 `deep-archive/` 目录（长期未访问）
- 🗑️ **删除**: 确认无价值后删除（需确认）
