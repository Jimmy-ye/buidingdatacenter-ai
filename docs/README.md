## 📝 文档维护

### 添加新文档

1. 确定文档属于哪个分类（00-04）
2. 将文档放置在对应目录下
3. 更新本索引文件

### 文档命名规范

- 使用中文命名，便于理解
- 文件名应清晰反映文档内容
- 日期在文档内容中说明，不在文件名中

### 过期文档处理

- 不再需要的文档移至 `archive/` 目录
- 或在文档开头添加 `[已废弃]` 标记
