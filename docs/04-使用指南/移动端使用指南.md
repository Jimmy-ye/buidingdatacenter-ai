## 1. 应用概述

### 1.1 功能介绍

BDC-AI 移动端是专为现场数据采集设计的移动应用，主要功能：

- 📷 **拍照上传**：快速采集设备铭牌、现场问题、仪表读数
- 🎤 **语音记录**：语音转文字，快速录入工程师备注
- 📝 **文字录入**：快速文字记录和表单填写
- 🌲 **工程结构浏览**：查看项目、建筑、系统、设备树形结构
- 🔍 **二维码扫描**：快速识别设备
- 📋 **资产查看**：查看设备关联的资产列表
- 💾 **离线操作**：无网络时也可正常使用，自动同步数据
- 📍 **GPS 定位**：记录采集地点

### 1.2 适用场景

**现场巡检**：
- 拍摄设备照片
- 记录设备状态
- 上报问题

**资产盘点**：
- 扫描设备二维码
- 拍摄设备铭牌
- 录入设备信息

**故障诊断**：
- 拍摄故障现场
- 语音描述问题
- 实时上传数据

**数据采集**：
- 仪表读数记录
- 现场测量记录
- 文档扫描

### 1.3 系统要求

**Android**：
- 系统版本：Android 7.0 (API 24) 及以上
- RAM：2 GB 及以上
- 存储空间：100 MB 可用空间
- 摄像头：支持
- 麦克风：支持
- GPS：支持（可选）

**iOS**：
- 系统版本：iOS 12.0 及以上
- 设备：iPhone 6s 及以上
- 存储空间：100 MB 可用空间

## 2. 安装和配置

### 2.1 安装应用

#### Android 安装

**方式 A：从应用商店安装（推荐）**

1. 打开 Google Play 商店
2. 搜索 "BDC-AI"
3. 点击 "安装"
4. 等待安装完成

**方式 B：从 APK 安装**

1. 下载 APK 文件（从企业官网或邮件链接）
2. 在手机上打开 "设置" → "安全"
3. 允许 "从未知来源安装应用"
4. 打开文件管理器，找到 APK 文件
5. 点击 APK 文件开始安装
6. 点击 "安装" → "打开"

#### iOS 安装

**方式 A：从 App Store 安装（推荐）**

1. 打开 App Store
2. 搜索 "BDC-AI"
3. 点击 "获取" 或下载图标
4. 验证 Face ID / Touch ID / 密码
5. 等待安装完成

**方式 B：TestFlight 安装（测试版）**

1. 从邮件获取 TestFlight 邀请链接
2. 点击链接打开 TestFlight
3. 点击 "安装"
4. 等待安装完成

### 2.2 权限说明

首次打开应用时，会请求以下权限：

| 权限 | 用途 | 是否必需 |
|-----|------|---------|
| 相机 | 拍照上传 | 必需 |
| 麦克风 | 语音录制 | 必需 |
| 存储 | 保存图片和缓存 | 必需 |
| 网络 | 上传数据 | 必需 |
| GPS | 记录位置 | 可选 |

**说明**：
- 这些权限仅用于核心功能
- 数据仅存储在应用沙盒中
- 不会访问通讯录、短信等隐私数据

### 2.3 配置服务器

#### 开发环境配置

**自动配置**（大多数情况）：

应用默认连接到开发服务器，无需配置。

**手动配置**：

1. 打开应用
2. 进入 "设置" → "服务器配置"
3. 输入服务器地址：
   ```
   http://192.168.1.100:8000
   ```
   或
   ```
   https://api.bdc-ai.com
   ```
4. 点击 "测试连接"
5. 连接成功后点击 "保存"

#### 生产环境配置

1. 打开应用
2. 进入 "设置" → "服务器配置"
3. 输入生产服务器地址
4. 点击 "测试连接"
5. 连接成功后点击 "保存"

**网络要求**：

- 移动端和服务器需要在同一网络，或
- 服务器有公网 IP，或
- 使用 VPN / Tailscale 虚拟局域网

## 3. 首次使用

### 3.1 注册和登录

#### 注册账号

**步骤**：

1. 打开应用
2. 在登录页面点击 "注册账号"
3. 填写注册信息：
   - 用户名（必填）
   - 邮箱（必填）
   - 密码（必填，至少 8 位）
   - 确认密码（必填）
4. 点击 "注册"
5. 注册成功后自动登录

**说明**：
- 注册需要管理员审核或使用邀请码（根据配置）
- 某些企业版本可能不支持自助注册

#### 登录

**步骤**：

1. 打开应用
2. 输入用户名和密码
3. （可选）勾选 "记住我" - 7 天内免登录
4. 点击 "登录"

**默认账号**（如适用）：

```
用户名：demo
密码：demo123
```

⚠️ **重要**：默认账号仅用于测试，请在正式环境使用自己的账号。

### 3.2 基本设置

#### 个人信息

**步骤**：

1. 登录后进入 "设置"
2. 点击 "个人信息"
3. 修改信息：
   - 显示名称
   - 邮箱地址
   - 手机号码
4. 点击 "保存"

#### 修改密码

**步骤**：

1. 进入 "设置" → "账号安全"
2. 点击 "修改密码"
3. 输入：
   - 当前密码
   - 新密码
   - 确认新密码
4. 点击 "确认修改"

#### 通知设置

**通知类型**：

- 上传成功通知
- 数据同步通知
- 系统公告通知
- 任务提醒通知

**配置步骤**：

1. 进入 "设置" → "通知设置"
2. 为每种通知类型选择：
   - 立即通知
   - 每日汇总
   - 关闭通知
3. 点击 "保存"

**缓存内容**：

- 工程结构数据
- 资产缩略图

**配置步骤**：

1. 进入 "设置" → "存储与缓存"
2. 查看缓存大小
3. 点击 "清除缓存" 释放空间

## 4. 核心功能

### 4.1 项目管理

#### 查看项目列表

**访问路径**：首页 → 项目列表

**显示内容**：

- 项目名称
- 客户名称
- 项目状态
- 项目类型

**筛选和排序**：

- 按状态筛选：全部/规划中/进行中/已完成
- 按类型筛选：全部/商业/工业/公共建筑
- 按名称搜索：输入关键词搜索

**操作**：

- 点击项目卡片查看详情
- 下拉刷新列表
- 上拉加载更多

#### 查看项目详情

**显示内容**：

- 项目基本信息（名称、客户、位置、类型、状态）
- 统计数据（建筑数量、设备数量、资产数量）
- 工程结构入口
- 资产列表入口

**操作**：

- 点击 "工程结构" 查看建筑、系统、设备
- 点击 "资产列表" 查看所有资产
- 点击 "刷新" 更新数据

### 4.2 资产采集

#### 拍照上传

**步骤**：

1. 进入 "资产采集" 页面
2. 点击 "📷 拍照上传" 按钮
3. 选择或填写：
   - 所属项目（必选）
   - 所属设备（可选）
   - 内容角色：现场问题/设备铭牌/仪表读数/一般资料
4. 点击 "📷 拍照" 或从相册选择
5. （可选）添加文字备注
6. （可选）录制语音备注
7. 点击 "上传"

**拍照技巧**：

- 保持相机稳定
- 确保光线充足
- 避免反光和阴影
- 设备铭牌：近距离拍摄，确保文字清晰
- 现场问题：拍摄整体场景，包含问题区域和周边环境

#### 添加备注

**文字备注**：

1. 拍照后点击 "添加备注"
2. 输入文字内容
3. 点击 "完成"

**语音备注**：

1. 点击 "🎤 语音" 按钮
2. 按住麦克风说话
3. 释放麦克风，自动转换为文字
4. 查看转换结果，手动编辑（如需要）
5. 点击 "完成"

**语音识别技巧**：

- 说话清晰，语速适中
- 避免环境噪音
- 使用专业术语时发音清晰
- 说完后稍等片刻再释放麦克风

#### 批量上传

**步骤**：

1. 进入 "资产采集" 页面
2. 点击 "📚 批量上传" 按钮
3. 从相册选择多张照片
4. 选择共同的所属项目和设备
5. 为每张照片添加备注（可选）
6. 点击 "批量上传"

**说明**：

- 批量上传会按顺序逐个处理
- 可以在后台继续上传
- 上传完成后会发送通知

### 4.3 工程结构

#### 查看工程结构树

**访问路径**：项目详情 → 工程结构

**显示方式**：

- 树形结构：Building → System → Device
- 可折叠/展开节点
- 显示设备数量

**操作**：

- 点击节点展开/折叠
- 长按节点显示菜单
- 点击设备查看详情

#### 查看设备详情

**显示内容**：

- 设备名称
- 设备类型
- 设备型号
- 额定功率
- 所属系统
- 所在区域
- 关联资产列表

**操作**：

- 查看关联资产
- 编辑设备信息（需要权限）
- 上传资产到此设备

#### 扫描二维码

**步骤**：

1. 进入 "工程结构" 页面
2. 点击 "🔍 扫描二维码" 按钮
3. 对准设备上的二维码
4. 自动识别并跳转到设备详情

**说明**：

- 需要相机权限
- 支持常见二维码格式
- 识别失败时手动输入设备 ID

### 4.4 数据同步

#### 在线模式

**特点**：

- 实时上传数据到服务器
- 实时从服务器获取最新数据
- 所有操作立即生效

**状态显示**：

- 绿色指示灯：已连接
- 数据实时同步

#### 离线模式

**特点**：

- 无网络时自动切换到离线模式
- 数据保存到本地 SQLite 数据库
- 可以正常查看缓存的数据
- 可以继续采集资产（保存在本地）

**状态显示**：

- 黄色指示灯：离线模式
- 显示缓存的数据量

**操作**：

- 查看缓存数据
- 继续采集资产
- 等待网络恢复后自动同步

## 5. 常见操作

### 5.1 现场巡检

#### 创建巡检记录

**步骤**：

1. 进入 "现场巡检" 页面
2. 选择巡检项目（如有）
3. 点击 "➕ 新建巡检"
4. 填写巡检信息：
   - 巡检时间
   - 巡检区域
   - 巡检人员
5. 开始巡检：
   - 拍摄设备照片
   - 记录设备状态
   - 上报发现的问题
6. 巡检完成后点击 "提交"

**巡检模板**（如配置）：

- HVAC 设备巡检
- 照明系统巡检
- 消防系统巡检
- 电梯巡检

#### 拍摄现场照片

**要求**：

- 拍摄设备整体外观
- 拍摄设备铭牌
- 拍摄异常部位特写
- 多角度拍摄（正面、侧面、背面）

**最佳实践**：

1. 先拍摄设备整体（了解环境）
2. 再拍摄铭牌（识别设备）
3. 最后拍摄问题部位（记录故障）

#### 填写检查表

**步骤**：

1. 选择检查项
2. 选择状态：正常/异常/需处理
3. 添加备注说明
4. （可选）上传照片
5. 点击 "保存"

**快捷操作**：

- 滑动切换状态
- 长按查看详细说明
- 批量标记（如全部正常）

### 5.2 问题反馈

#### 上报问题

**步骤**：

1. 在巡检或资产采集中发现问题
2. 点击 "🚩 上报问题" 按钮
3. 填写问题信息：
   - 问题类型（设备故障/安全隐患/能效问题/其他）
   - 严重程度（紧急/重要/一般）
   - 问题描述
   - 现场照片（已拍摄）
4. 点击 "提交"

**问题类型**：

- 设备故障：设备无法正常运行
- 安全隐患：可能造成安全风险的问题
- 能效问题：能耗异常、能源浪费
- 其他：需要说明的问题

**问题跟踪**：

1. 进入 "问题反馈" 页面
2. 查看上报的问题列表
3. 点击问题查看处理进度
4. 查看处理结果和反馈

## 6. 数据同步

### 6.1 在线模式

#### 自动同步

**触发条件**：

- 应用启动时
- 切换回应用时
- 数据变更时
- 定期同步（每 5 分钟）

**同步内容**：

- 工程结构数据
- 资产元数据
- 用户信息
- 系统配置

**同步指示器**：

- 🟢 已同步：数据最新
- 🔄 同步中：正在更新
- ⚠️ 同步失败：检查网络连接

#### 手动同步

**步骤**：

1. 下拉任意列表页面
2. 等待加载图标消失
3. 数据更新完成

**或**：

1. 进入 "设置" → "数据同步"
2. 点击 "立即同步" 按钮
3. 等待同步完成

### 6.2 离线模式

#### 自动切换

当网络不可用时，应用自动切换到离线模式：

- 🟡 离线模式指示器
- 可以查看缓存的数据
- 可以继续采集资产
- 数据保存在本地

#### 离线操作

**可以做的操作**：

- 查看缓存的项目和工程结构
- 拍照上传（保存到本地）
- 录入文字记录
- 语音转文字

**不能做的操作**：

- 查看未缓存的数据
- 下载新的资产
- 调用 AI 分析（需要网络）

#### 恢复在线

**自动同步**：

1. 连接到网络
2. 应用检测到网络可用
3. 自动开始同步本地数据
4. 同步完成后切换到在线模式

**手动同步**：

1. 进入 "设置" → "数据同步"
2. 查看待同步数据数量
3. 点击 "立即同步"
4. 等待同步完成

### 6.3 冲突处理

#### 冲突检测

**冲突场景**：

- 离线时修改了数据，服务器上也被修改
- 多人同时修改同一设备
- 数据版本不匹配

**冲突处理策略**：

1. **服务器优先**（默认）：
   - 保留服务器数据
   - 本地修改被覆盖
   - 通知用户

2. **本地优先**：
   - 保留本地修改
   - 覆盖服务器数据
   - 通知其他用户

3. **手动合并**：
   - 显示冲突内容
   - 用户选择保留哪个版本
   - 或手动合并数据

#### 查看冲突

**步骤**：

1. 进入 "设置" → "数据同步"
2. 查看 "同步冲突" 列表
3. 点击冲突项查看详情
4. 选择处理方式：
   - 保留服务器版本
   - 保留本地版本
   - 手动编辑

## 7. 故障排查

### 7.1 连接问题

#### 无法连接服务器

**症状**：应用显示 "无法连接到服务器"

**可能原因**：
1. 网络连接问题
2. 服务器地址配置错误
3. 服务器未运行
4. 防火墙阻止连接

**解决方案**：

1. **检查网络连接**：
   - 打开浏览器，访问其他网站
   - 检查 WiFi 或移动数据是否开启
   - 尝试切换网络（WiFi ↔ 移动数据）

2. **检查服务器配置**：
   - 进入 "设置" → "服务器配置"
   - 确认服务器地址正确
   - 点击 "测试连接"

3. **检查服务器状态**：
   - 联系管理员确认服务器运行正常
   - 尝试在浏览器中访问服务器地址

4. **检查 VPN 设置**：
   - 如果使用 VPN，确保连接正常
   - 或尝试断开 VPN

#### Token 失效

**症状**：频繁被要求重新登录

**可能原因**：
1. Token 过期（默认 7 天）
2. 服务器重启导致 Token 失效
3. 多设备登录，Token 被覆盖

**解决方案**：

1. **重新登录**：
   - 退出登录
   - 重新输入用户名和密码
   - 勾选 "记住我"

2. **检查设备数量**：
   - 如果在多台设备登录，可能相互覆盖
   - 建议在主要设备使用

### 7.2 功能问题

#### 上传失败

**症状**：上传进度卡住或显示上传失败

**可能原因**：
1. 文件过大（超过 50 MB）
2. 网络连接中断
3. 服务器存储空间不足
4. 服务器内部错误

**解决方案**：

1. **压缩图片**：
   - 使用相机设置降低分辨率
   - 或使用图片压缩工具

2. **检查网络**：
   - 切换到更稳定的网络
   - 或在 WiFi 环境下上传

3. **分批上传**：
   - 不要一次上传过多文件
   - 分批上传，每次 5-10 张

4. **查看上传历史**：
   - 进入 "设置" → "上传历史"
   - 查看失败原因
   - 重新上传失败的文件

#### 语音识别失败

**症状**：语音转文字不准确或失败

**可能原因**：
1. 环境噪音过大
2. 说话不清晰
3. 使用方言或专业术语
4. 麦克风权限未授予

**解决方案**：

1. **改善环境**：
   - 减少背景噪音
   - 在安静环境中录制

2. **调整说话方式**：
   - 说话清晰，语速适中
   - 避免使用方言
   - 专业术语可手动编辑

3. **检查权限**：
   - 进入系统设置
   - 确认应用有麦克风权限

4. **手动编辑**：
   - 语音转换后手动修正错误
   - 或直接使用文字输入

### 7.3 性能问题

#### 应用卡顿

**症状**：操作响应慢，界面卡顿

**可能原因**：
1. 缓存数据过多
2. 应用版本过旧
3. 手机性能不足
4. 后台运行的应用过多

**解决方案**：

1. **清除缓存**：
   - 进入 "设置" → "存储与缓存"
   - 点击 "清除缓存"
   - 重新登录

2. **更新应用**：
   - 检查应用商店更新
   - 更新到最新版本

3. **重启应用**：
   - 完全关闭应用（从后台清除）
   - 重新打开

4. **重启手机**：
   - 关机后重新启动
   - 清理后台应用

#### 占用空间大

**症状**：应用占用大量存储空间

**可能原因**：
1. 缓存的图片过多
2. 离线数据占用空间
3. 应用日志文件过大

**解决方案**：

1. **清除缓存**：
   - 进入 "设置" → "存储与缓存"
   - 查看缓存大小
   - 点击 "清除缓存"

2. **删除离线数据**：
   - 进入 "设置" → "离线数据"
   - 删除不需要的离线项目

3. **卸载重装**（最后手段）：
   - 备份数据
   - 卸载应用
   - 重新安装

#### 电量消耗快

**可能原因**：
1. 频繁数据同步
2. GPS 定位开启
3. 后台运行
4. 屏幕常亮

**解决方案**：

1. **调整同步频率**：
   - 进入 "设置" → "数据同步"
   - 增加同步间隔

2. **关闭 GPS**：
   - 如果不需要定位，关闭 GPS

3. **关闭后台刷新**：
   - 进入系统设置
   - 关闭应用后台刷新

4. **调整屏幕亮度**：
   - 降低屏幕亮度
   - 缩短屏幕超时时间

- 可拍摄多张（不同时间）

- 包含关键信息（时间、数量、型号）
- 避免模糊描述

**语音备注**：

- 直接说重点
- 避免冗余信息

## 9. FAQ

### 9.1 常见问题

**Q1: 应用支持哪些手机型号？**

A: Android 7.0+ 或 iOS 12.0+ 的手机均可使用。推荐使用 3 年内发布的机型以获得最佳性能。

**Q2: 离线模式可以保存多少数据？**

A: 本地缓存限制为 500 MB，约可保存 1000-2000 张照片和数百条记录。可以在设置中查看缓存大小。

**Q3: 可以同时登录多个设备吗？**

A: 可以，但新设备登录会使旧设备的 Token 失效。建议主要使用一个设备，或定期重新登录。

**Q4: 照片会上传到云端吗？**

A: 照片上传到 BDC-AI 服务器，存储在企业私有云或自建服务器中，不会上传到公共云。

**Q5: 如何保证数据安全？**

A: 所有数据传输使用 HTTPS 加密，服务器存储使用加密，访问需要权限认证。建议设置手机锁屏密码。

**Q6: 语音识别支持哪些语言？**

A: 当前支持中文普通话识别。其他语言和方言支持将在后续版本添加。

**Q7: 可以导出数据到其他应用吗？**

A: 支持。可以导出 PDF 报告和 Excel 表格，并通过邮件、微信等方式分享。

**Q8: 应用消耗多少流量？**

A: 正常使用每天约 10-50 MB（取决于上传照片数量）。建议在 WiFi 环境下上传大量数据。

**Q9: 如何联系技术支持？**

A: 在应用内进入 "设置" → "帮助与反馈"，或发送邮件到 support@bdc-ai.com。

**Q10: 可以离线使用 AI 分析功能吗？**

A: 不可以。AI 分析需要调用服务器 API，需要网络连接。可以先拍照，网络恢复后再分析。
